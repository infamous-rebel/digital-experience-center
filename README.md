# Studio Z Digital Experience Center

A living virtual immersive venue showcasing Studio Z's capabilities in projection mapping, holography, LED experiences, and more.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/infamous-rebel/digital-experience-center.git
   cd digital-experience-center
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Add 3D libraries:
   ```bash
   npm install three @react-three/fiber @react-three/drei gsap framer-motion
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) to explore the virtual experience center.

## Tech Stack
- Next.js 15 + React 19
- Three.js + React Three Fiber
- Tailwind CSS
- GSAP for animations
- Spatial Audio (Web Audio API)

Built as a digital twin of a premium physical experience center.
